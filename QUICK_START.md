### 1. Install Dependencies
```bash
npm install
```

### 2. Run Development Server
```bash
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to see your portfolio!

### 3. Customize Your Content

Edit these files to personalize your portfolio:

- **Hero Section**: `components/Hero.tsx` - Change name, roles, resume link
- **About**: `components/About.tsx` - Update your bio
- **Experience**: `components/Experience.tsx` - Add your work experience
- **Tech Stack**: `components/TechStack.tsx` - Update technologies
- **Projects**: `components/Projects.tsx` - Add your projects
- **Education**: `components/Education.tsx` - Update education details
- **Awards**: `components/Awards.tsx` - Add your awards

### 4. Add Your Resume

1. Place your resume PDF in the `public/` folder
2. Name it `resume.pdf`
3. The "Download Resume" button will automatically link to it
